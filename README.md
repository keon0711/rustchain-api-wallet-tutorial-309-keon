# RustChain API and Wallet Tutorial

Submission artifact for RustChain issue #309.

Video: `rustchain-api-wallet-tutorial-309.mp4`

What it demonstrates:

- live `https://rustchain.org/health`
- `tools/rustchain_wallet_cli.py --help`
- `tools/rustchain_wallet_cli.py epoch`
- active miner summary from `tools/rustchain_wallet_cli.py miners`
- read-only wallet balance, history, and lottery eligibility checks

The tutorial uses the public receiving wallet only. It does not show private keys, seed phrases, or signed transfer commands.
